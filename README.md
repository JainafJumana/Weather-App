# 🌦️ Weather App

## 🔗 Demo
 https://jainafjumana.github.io/Weather-App/


## 📖 About
This is a simple Weather App built using HTML, CSS, and JavaScript. It uses the OpenWeather API to display real-time weather information based on the city entered by the user.

## ✨ Features
- Search weather by city name
- Display temperature
- Display humidity
- Display wind speed
- Dynamic weather icons
- Invalid city name error handling
- Search using Enter key or Search button

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## 🚀 How to Use
1. Enter a city name.
2. Click the Search button or press **Enter**.
3. View the current weather details.
